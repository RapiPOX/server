## Getting started

### Setup environment

Set proper nvm version

```bash
nvm use
```

Install dependencies

```bash
pnpm i
```

Copy .env.example to .env

```bash
cp .env.example .env
```

Edit .env file with proper values

### Run in developer mode

```
pnpm dev
```
